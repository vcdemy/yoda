# YODA

YODA是<ins>**YO**</ins>LO <ins>**DA**</ins>TA Labeler的縮寫。

它是一個使用rect box來簡單的標註YOLO資料的工具。

我們主要使用它來作為訓練自己的YOLO模型的教學工具。

網址： https://vcdemy.github.io/yoda

* [makesense.ai](https://github.com/SkalskiP/make-sense)
